☕ Café website – Responsive Bootstrap Card Layout

📌 Project Overview
This project is a responsive Café Menu web page developed using HTML5, CSS3, and Bootstrap 5.
It showcases different hot and cold coffee beverages using Bootstrap card components with hover effects and responsive grid layout.
The project demonstrates effective use of Bootstrap grid system, card components, and custom CSS styling.

🚀 Features
Responsive card-based layout
Bootstrap 5 Grid System
Hover animation effect on cards
Attractive background image
Google Fonts integration
Clean UI design
Mobile-first responsive approach

☕ Menu Items Included
Hot Beverages
Hot Cappuccino
Hot Americano
Hot Latte
Hot Espresso & Chocolate
Cold Beverages
Iced Cappuccino
Iced Americano
Iced Strawberry Latte
Iced Mocha

🛠️ Technologies Used
HTML5
CSS3
Bootstrap 5.3
Google Fonts
Bootstrap Grid System
Bootstrap Card Component

💡 UI Enhancements
Smooth card hover lift effect using:
transform: translateY(-20px);
transition: 0.6s ease;
Background image with full-screen cover
Custom font styling using Google Fonts
Hover background color change on card body

📂 Project Structure
index.html   – Structure
style.css    – Styling
